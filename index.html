Welcome! This API lets you fetch YouTube search results without an API key.

Example Usage

Search for "adhenti gani":

https://railway.app/api/search?q=adhenti+gani

With pagination (pageToken):

https://railway.app/api/search?q=adhenti+gani&pageToken=XYZ123

Parameters

  • q (required) – Search query
  • pageToken (optional) – Pass this to fetch the next page

Response is JSON with video results, thumbnails, titles, and more.
